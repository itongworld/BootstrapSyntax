在Bootstrap主要提供了三种代码风格：

1、使用<code></code>来显示单行内联代码,<code>一般是针对于单个单词或单个句子的代码；
2、使用<pre></pre>来显示多行块代码,<pre>一般是针对于多行代码（也就是成块的代码）；
3、使用<kbd></kbd>来显示用户输入代码,<kbd>一般是表示用户要通过键盘输入的内容；
